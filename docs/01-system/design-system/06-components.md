# Komponen (Components)

Daftar komponen UI utama yang dapat digunakan kembali (reusable) di seluruh aplikasi.

## 1. Buttons (Tombol)
- **Primary Button**: Warna latar `Primary 500`, teks putih, lengkungan `radius-sm`.
- **Secondary Button**: Warna latar transparan/`Primary 50`, border `Primary 500`, teks `Primary 500`.
- **Danger Button**: Warna latar `Danger`, teks putih.
- **Text Button**: Hanya teks tanpa latar/border, warna `Primary 500` atau `Grey 500`.

## 2. Inputs (Masukan)
- **Text Field**: Border `Grey 300`, padding `md` (16px), background putih, fokus border `Primary 500`.
- **Dropdown**: Sama dengan text field, dilengkapi ikon chevron di sebelah kanan.

## 3. Cards (Kartu)
- **Standard Card**: Background putih, lengkungan `radius-md`, bayangan halus (elevation 2), padding internal `md`. Digunakan untuk Job Pool, daftar Campaign, riwayat transaksi.

## 4. Modals & Dialogs
- **Bottom Sheet**: Digunakan untuk filter, menu opsi (mobile-first). Lengkungan atas `radius-lg`.
- **Alert Dialog**: Pop-up di tengah layar untuk konfirmasi aksi destruktif.

## 5. Badges (Label Status)
- Komponen kecil untuk menampilkan status. Lengkungan `radius-sm`, teks `Label Small`. (Contoh: `Pending` warna Warning, `Valid` warna Success).
