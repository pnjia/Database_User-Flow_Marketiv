# Spasi (Spacing)

Sistem spasi digunakan untuk margin, padding, grid, dan tata letak secara keseluruhan. Gunakan nilai ini secara konsisten agar elemen terlihat sejajar.

## Skala Spasi (Spacing Scale)
| Token | Ukuran | Penggunaan Utama |
|---|---|---|
| `xs` | 4px | Jarak antar icon dan teks kecil |
| `sm` | 8px | Padding kecil, jarak antar elemen terkait |
| `md` | 16px | Padding standar (kontainer, card), margin antar komponen |
| `lg` | 24px | Jarak antar section yang berbeda |
| `xl` | 32px | Jarak besar untuk layout utama |
| `xxl` | 48px | Jarak pemisah super besar (biasanya untuk halaman landing) |

## Border Radius (Lengkungan)
| Token | Ukuran | Penggunaan |
|---|---|---|
| `radius-sm` | 8px | Input field, button kecil, label |
| `radius-md` | 12px | Card reguler, dialog modal kecil |
| `radius-lg` | 16px | Bottom sheet, kontainer besar |
