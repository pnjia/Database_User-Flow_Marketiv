# Warna (Colors)

Panduan warna untuk aplikasi Marketiv. 

## Palet Warna Utama (Primary Colors)
- **Primary 50**: `#FFF7ED` (Background ringan)
- **Primary 100**: `#FFEDD5` (Hover state)
- **Primary 500**: `#F97316` (Brand utama - Oranye)
- **Primary 600**: `#EA580C` (Primary hover/active)
- **Primary 700**: `#C2410C` (Primary dark)

## Palet Warna Sekunder (Secondary Colors)
- **Secondary 500**: `#1E3A5F` (Navy / Biru Gelap)
- **Secondary 600**: `#162D4A` 
- **Secondary 700**: `#0F2039`

## Semantic Colors
Digunakan untuk memberikan makna pada komponen antarmuka.
- **Success**: `#16A34A` (Hijau - Berhasil, selesai)
- **Warning**: `#D97706` (Kuning/Oranye - Peringatan, tertunda)
- **Danger**: `#DC2626` (Merah - Gagal, error, hapus)
- **Info**: `#2563EB` (Biru - Informasi, petunjuk)

## Netral (Neutrals)
- **Grey 50**: `#F9FAFB` (Background aplikasi)
- **Grey 100**: `#F3F4F6` (Background komponen)
- **Grey 300**: `#D1D5DB` (Borders, divider)
- **Grey 500**: `#6B7280` (Teks sekunder, placeholder)
- **Grey 700**: `#374151` (Teks utama standar)
- **Grey 900**: `#111827` (Teks judul tebal)
