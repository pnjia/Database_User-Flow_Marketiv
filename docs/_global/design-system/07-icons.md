# Ikon (Icons)

Panduan penggunaan ikon dalam aplikasi Marketiv.

## Sistem Ikon
- Kami menggunakan ikon bergaya **outline** (garis) untuk elemen yang tidak aktif, dan bergaya **solid/filled** untuk elemen yang sedang aktif (contoh: Bottom Navigation).
- Ikon utama menggunakan pendekatan mirip **Cupertino Icons** (iOS style) atau **Material Rounded** agar terlihat modern.

## Ukuran Ikon (Icon Sizes)
- **Small (sm)**: `16px` (Ikon dalam badge, keterangan tambahan)
- **Medium (md)**: `24px` (Standar untuk aksi utama, tombol ikon, Bottom Nav)
- **Large (lg)**: `32px` (Ikon ilustrasi, status empty state)

## Praktik Terbaik (Best Practices)
- Selalu sandingkan ikon ambigu dengan teks penjelas (Label).
- Jaga konsistensi ketebalan garis ikon (biasanya 1.5px hingga 2px).
- Sesuaikan warna ikon dengan skala warna (contoh: ikon error gunakan warna `Danger`).
